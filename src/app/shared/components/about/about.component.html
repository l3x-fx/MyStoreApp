About MyStore

Allow me to introduce my latest Full Stack project. MyStore is a showcase of my skills in FullStack development. It was built using a modern technology stack:

  • On the front end, I utilized TypeScript and Angular to create a dynamic and responsive user interface.
  • For state management, I employed NgRx to ensure an efficient data flow throughout the application.
  • On the back end, I harnessed Node.js and Express to develop a robust and performant API.
  • The application's data is stored and managed in a PostgreSQL relational database, ensuring data integrity and reliability.

MyStore encapsulates my dedication to creating visually captivating and functional websites to ensure an enjoyable user experience.
